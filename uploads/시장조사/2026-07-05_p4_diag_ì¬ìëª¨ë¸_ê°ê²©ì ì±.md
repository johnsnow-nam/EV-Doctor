# P4 Diagnostic Tester 사업모델 및 가격 정책

작성: 2026-07-05 KST

대상: `boards/p4_diag` 제품화/영업 자료

## 1. 결론

P4는 충전기가 아니라 **EV처럼 행동해서 충전기를 시험하는 전문 진단 테스트기**다.

따라서 가격은 단순 CP/PP 어댑터 시장이 아니라, 아래 중간 포지션으로 잡아야 한다.

```text
CP/PP 어댑터             P4 Diagnostic Tester             연구소급 분석기
50만~170만원대      ->   330만~490만원대            ->   1천만원~수천만원 이상
```

권장 가격:

| 패키지 | 권장가 | 용도 |
|---|---:|---|
| P4 Field Pilot | 3,300,000원 VAT 별도 | 초기 10대, 현장 검증 고객 |
| P4 Pro | 4,900,000원 VAT 별도 | 정식 판매 기준 |
| P4 Pro + Field Support | 6,900,000~8,900,000원 VAT 별도 | 충전기 제조사/운영사 프로젝트 |

## 3. P4의 기능 포지션

### 3.1 상세 기능 사양

P4는 아래 기능을 한 장비 안에서 제공하는 것을 목표로 한다.

| 영역 | 기능 사양 | 고객 가치 |
|---|---|---|
| CP/PP/PWM | CP State A/B/C 표시, CP 전압, PWM duty, PP 체결 상태 | 충전기 기본 제어 신호 이상을 즉시 분류 |
| 릴레이/AC 출력 | K1, K2, AC output 상태 표시 | 충전기가 실제 출력을 내는지와 릴레이 전이를 확인 |
| PLC/SLAC | SLAC matched, EVSE MAC, PLC link 진행 상태 | 차량 없이 PLC 계층 문제를 현장에서 확인 |
| SDP | EVSE IPv6 address, TCP port, SDP 결과 | V2G 연결 전 네트워크 발견 단계 확인 |
| ISO 15118 AC | ChargingStatus, EV SoC, EVSE max current, V2G report, error count | 충전 시작/유지 단계의 통신 정상 여부 확인 |
| CPTEST | 충전 중 CP/PWM/K1/K2/AC 확인, B/C 전이 시험 | 충전 중 충전기 반응을 안전한 범위에서 재검증 |
| UI | LCD, touch, rotary, BACK/STOP/RETEST | 현장 엔지니어가 노트북 없이 사용 |
| 안전/복구 | HOME/RETEST reset, relay OFF, transient fault 지연 판정 | 반복 시험 중 찌꺼기 상태와 오탐을 줄임 |
| 로그/리포트 | 화면 report와 로그 report 의미 통일 | 고객/개발팀 간 장애 재현과 전달 비용 절감 |
| PC 분석·증적 도구 | live log, LCD capture PNG, mock 회귀, 충전기별 fixture, GUI 로그/PIB 분석 | 현장 판단을 고객사 품질팀·개발팀에 전달 가능한 증적으로 패키지화 |

### 3.1.1 PC toolchain의 제품 의미

`/tools`는 개발 보조 폴더가 아니라 P4 제품의 PC-side evidence toolchain으로 포지셔닝할 수 있다. 본체가 현장에서 CP/PLC/V2G 상태를 보여 준다면, PC toolchain은 그 순간을 로그·화면·반복 시험 결과로 남겨 고객사에 설명 가능한 리포트로 만든다.

| 도구 영역 | 대표 기능 | 사업적 의미 |
|---|---|---|
| Live console / log capture | 충전기별 이름표를 붙여 실시간 콘솔 로그를 `tools/logs`에 저장 | 현장 엔지니어와 개발팀이 같은 파일을 보며 장애 지점을 공유 |
| Screen capture | LCD 화면을 PNG로 저장하고 메뉴/오류 화면을 자동 캡처 | 리포트에 실제 화면 증거를 첨부해 고객 설득력 강화 |
| 자동 회귀 시험 | mock 재생, 반복 reset, CP 전이, 실충전 시작 검증 | 펌웨어 업데이트 후 기존 GoodCase가 깨지지 않았는지 확인 |
| 충전기별 fixture | 한 번 성공한 세션을 녹화해 재생 가능한 테스트 데이터로 변환 | 한솔, A사, B사 충전기 차이를 재현 가능한 호환성 자료로 축적 |
| GUI / PIB / PCAP 분석 | GUI 로그 뷰어, QCA PIB 확인, PCAP·캡처 산출물 관리 | 현장 지원과 연구소 분석 사이를 이어 주는 PC 분석 환경 제공 |

따라서 P4는 "휴대형 본체"만이 아니라 "본체 + PC 증적 도구 + 충전기별 호환성 데이터 축적"으로 팔아야 한다.

### 3.2 기능 비교 매트릭스

| 기능 | Fluke/Megger급 AC 어댑터 | P4 Diagnostic Tester | Keysight/Comemso급 연구소 장비 |
|---|---:|---:|---:|
| CP/PP 상태 시뮬레이션 | O | O | O |
| CP 전압/PWM duty 본체 표시 | △ | O | O |
| K1/K2/AC 출력 상태 표시 | △ | O | O |
| PLC/SLAC matched 표시 | X | O | O |
| SDP IPv6/port 표시 | X | O | O |
| ISO 15118 AC ChargingStatus 표시 | X | O | O |
| EV SoC/EVSE max current 표시 | X | O | O |
| PC 로그/화면 캡처/자동 회귀 도구 | X | O | O |
| 충전 중 CPTEST/B-C 전이 시험 | △ | O | O |
| 현장 휴대성과 즉시 판독 | O | O | △ |
| 인증·연구소급 conformance | X | X | O |

해석:

- 단순 AC 어댑터는 전기적 접속 상태 확인에는 좋지만, PLC/SLAC/SDP/ISO15118 문제를 직접 설명하지 못한다.
- 연구소급 장비는 기능이 가장 넓지만 가격과 운용 난도가 높아 설치/A/S 현장의 상시 휴대 장비로는 부담이 크다.
- P4는 두 시장 사이에서 "충전 실패 원인 분류"에 필요한 핵심 값을 현장 화면으로 제공한다.

### 3.3 비교 우위

P4의 비교 우위는 "최고급 인증 장비"가 아니라 "현장 문제 해결 속도"다.

| 비교 대상 | P4 우위 | 설명 |
|---|---|---|
| Fluke/Megger/Metrel adapter | 통신 상태까지 확인 | CP/PP만이 아니라 SLAC, SDP, V2G AC 상태를 보여준다. |
| 일반 오실로스코프/멀티미터 조합 | 사용 절차 단순화 | 현장 엔지니어가 여러 장비를 조합하지 않아도 된다. |
| 실제 차량 테스트 | 반복성과 통제성 | EV SoC/CPTEST/STOP/RETEST 흐름을 장비가 통제한다. |
| 연구소급 분석기 | 가격·휴대성 | 수천만원대 장비 없이 1차 원인 분류가 가능하다. |
| 충전기 업체 내부 로그만 보는 방식 | 고객/현장 공통 언어 | 충전기 밖에서 CP/PLC/V2G 상태를 독립적으로 관찰한다. |

P4가 단순 어댑터보다 비싸야 하는 이유:

- 화면이 있는 독립형 장비다.
- CP/PP/PWM만 보지 않고, K1/K2/AC 출력 상태를 같이 보여준다.
- PLC/SLAC/SDP 진행 상태를 표시한다.
- ISO 15118 AC ChargingStatus, EV SoC, EVSE max current, V2G report를 표시한다.
- 충전 중 CPTEST로 B/C 전이와 릴레이 반응을 확인할 수 있다.
- HOME/RETEST/STOP/reset 흐름이 있고, 실충전 중 1분 demo cap 제거 및 실제 충전 유지 검증을 했다.
- 개발 콘솔 위험 명령 잠금, 입력 debounce/event, 화면 깜빡임 완화 등 현장 사용성을 반영했다.

P4가 연구소급 장비보다 싸야 하는 이유:

- 고전압/대전류 부하뱅크가 아니다.
- 정식 conformance 인증 장비가 아니다.
- AC ISO 15118 현장 진단 중심이다.
- DC/CCS 전체 인증, ISO 15118-20, Plug & Charge PKI 검증을 모두 보장하는 장비가 아니다.
- VAS/BIP는 지원 충전기에서 별도 검증한다. VAS 미지원 충전기의 `no_response`는 실패가 아니라 정상 관측값이다.
- PC toolchain은 현장 증적과 반복 시험을 제공하지만, 연구소급 conformance 자동화 전체를 대체하지는 않는다.

## 4. 팔 때의 제품 문구

권장 제품명:

```text
P4 Diagnostic Tester
EVSE Field Diagnostic Tester for CP/PWM/PLC/ISO15118 AC
```

고객 설명:

```text
P4 Diagnostic Tester는 충전기를 EV처럼 물려 보고,
CP/PWM/PP 전기 신호와 SLAC/SDP/ISO15118 AC 통신 흐름을 한 화면에서 확인해
설치·개발·A/S 현장의 문제 원인을 빠르게 좁혀 주는 전문 진단 장비입니다.
```

반드시 빼야 할 과장:

- "인증 장비"
- "계량기/과금 검증 장비"
- "모든 충전기 100% 호환"
- "DC 급속 충전기 인증 가능"
- "VAS/BIP 완전 지원"

권장 표현:

- "현장 진단용"
- "충전 시작/유지/종료 흐름 확인"
- "CP/PP/PWM 및 ISO 15118 AC 상태 표시"
- "충전기 개발·설치·A/S 지원"
- "충전 실패 원인 분류"

고객용 HTML 자료에는 아래 시각 자료를 포함한다.

| 시각 자료 | 목적 |

이 시각 자료의 목적은 "P4가 더 비싼 어댑터"가 아니라 "현장 문제 원인을 분해해 주는 진단기"라는 인식을 만드는 것이다.

## 5. 가격이 이렇게 형성되어야 하는 이유

### 5.1 100만원대는 안 맞다

100만원대는 Fluke/Megger/Metrel 계열 AC EVSE adapter의 시장이다.

이 가격대 제품은 보통:

- EV 연결 저항 시뮬레이션
- CP/PP 상태 선택
- PE/CP error
- 외부 계측기 연결 단자

중심이다.

P4를 100만원대에 놓으면 고객은 P4를 "조금 복잡한 어댑터"로 인식한다. 그러면 PLC/SLAC/ISO15118 화면, 펌웨어, 로그, 실충전 검증의 가치가 가격에 반영되지 않는다.

### 5.2 300만원대는 파일럿 진입가로 적절하다

330만원은 고객 입장에서 단순 어댑터보다 비싸지만, 연구소급 장비보다 훨씬 낮다.

초기 고객에게 설명하기 좋다.

```text
고급 분석기까지 살 필요는 없지만,
단순 CP 어댑터로는 통신 문제가 안 보이는 현장용 장비입니다.
```

파일럿 조건:

- 초기 10대 한정
- 현장 로그 공유 조건
- 충전기 호환성 피드백 조건
- VAS/BIP beta 고지. VAS 미지원 충전기에서는 `no_response`가 정상 결과임을 설명
- 6개월 펌웨어 업데이트 포함
- 기본 PC 캡처 도구 포함

### 5.3 490만원은 정식 제품가로 적절하다

P4 Pro 490만원은 다음 가치를 포함한다.

- 독립형 LCD/터치/로터리 장비
- EVCC 역할 시뮬레이션
- CP/PP/PWM/K1/K2/AC 상태 표시
- SLAC/SDP/V2G report 표시
- 충전 중 CPTEST
- PC evidence toolchain: live log, LCD capture, mock 회귀, 충전기별 fixture
- 로그 추출과 고객 리포트 증적화
- 1년 펌웨어 업데이트
- 기본 기술지원

이 가격은 Fluke/Megger adapter보다 높고, Comemso/Keysight 계열보다 낮다. 즉, P4의 실제 위치인 "현장형 통신 진단기"를 표현한다.

### 5.4 690만~890만원은 지원 포함 가격이다

충전기 제조사/운영사 납품은 장비만 파는 구조가 아니다.

실제 비용은 다음에서 발생한다.

- 고객 충전기별 SLAC/SDP/ISO15118 호환성 확인
- 현장 로그 분석
- PC toolchain 기반 화면 캡처·fixture·회귀 리포트 작성
- 펌웨어 패치
- 고객사 엔지니어 교육
- 장애 리포트 작성
- 특정 충전기 화면/로그 항목 보강

따라서 "P4 Pro + Field Support"는 장비 가격이 아니라 프로젝트 초기 지원 비용까지 포함한 패키지로 팔아야 한다.

## 7. 원가와 마진 관점

초기 소량 생산에서는 BOM만 보고 가격을 정하면 안 된다.

가격에 반영해야 할 항목:

- PCB/부품/커넥터/케이스/케이블
- 조립·검수·플래시·캘리브레이션
- 포장·문서·출고 테스트
- 초기 불량/warranty reserve
- 펌웨어 유지보수
- 고객 충전기 호환성 대응
- 현장 로그 분석
- 다음 하드웨어 revision 비용

P4는 하드웨어보다 펌웨어/통신/현장 문제 해결 가치가 큰 제품이다. 따라서 원가 배수 방식보다 "고객이 절약하는 시간" 기준으로 가격을 잡아야 한다.

현장 엔지니어 2명이 충전기 문제를 하루 못 잡으면 인건비와 출장비만 수십만~수백만원이 든다. P4가 CP/PLC/ISO15118 원인을 10~30분 안에 좁혀 주면 490만원 가격은 B2B 고객에게 방어 가능하다.
